🧬 DNA Sequence Analysis Tool (BioPython)
Overview

This project is a simple DNA analysis tool built using Python and BioPython. It performs basic computational biology operations on a user-provided DNA sequence.

Features
DNA complement and reverse complement generation
DNA to RNA transcription
Nucleotide counting (A, T, G, C)
Protein translation
GC content calculation
Motif search (CGT) detection
Technologies Used
Python
BioPython (Seq, gc_fraction)
How It Works

The program takes a DNA sequence as input and processes it using BioPython’s sequence tools to generate biological insights such as RNA conversion, protein translation, and motif location.

Output Includes
Complementary DNA strand
Reverse complement strand
RNA sequence
Nucleotide composition
Protein sequence
GC content percentage
Motif position
Note

This tool is intended for educational and introductory bioinformatics analysis.
